Projects

Python Raycaster

For my computing A-level coursework, I created a raycasting project in python using the Pygame package and assets found online. Designed to replicate the rendering technique of Wolfenstein3D but have the visual style of DOOM 1993, my program has a unique visual design with customisable maps, modes and screen size.Whilst starting as a simple raycasting environment, I later added sprites, randomly generated maps, interactable areas of the map and more. This project achieved me nearly full marks for my coursework.

Isometric Project

This was a simple Isometric game with a building and deleting mechanic, made for a school project. The map is generated from information provided by the user. The game has a fully controllable camera, controlled with WASD, and then the player moves with the arrow keys. Other controls include:

    To build - left click
    To move up - space bar
    To move down - shift key
    To swap material - press 1, 2 or 3
